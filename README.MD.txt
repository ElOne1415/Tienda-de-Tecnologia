# Tienda de Tecnología

Proyecto web estático desarrollado con **HTML5, CSS3 y JavaScript**.

Este sitio simula la página oficial de una tienda de tecnología, incluyendo catálogo de productos, información institucional y formulario de contacto.

---

##  Objetivo del Proyecto

El objetivo es aplicar buenas prácticas de desarrollo frontend:

- Separación de responsabilidades (HTML / CSS / JS)
- Organización profesional de carpetas
- Uso básico de control de versiones con Git
- Estructura lista para escalar a proyectos más grandes

---
##  Justificación de la Estructura

La estructura del proyecto sigue un esquema :

###  HTML en la raíz
Los archivos HTML están en la raíz porque:
- Es un proyecto estático pequeño/mediano.
- Facilita la navegación directa.
- Es estándar en proyectos frontend sin backend.

### Carpeta `css/`
Contiene todos los estilos.
- Evita mezclar estilos dentro del HTML.
- Permite mantener el diseño separado de la estructura.

###  Carpeta `js/`
Contiene la lógica JavaScript.
- Mantiene el comportamiento separado del diseño.
- Facilita mantenimiento y escalabilidad.

###  Carpeta `assets/`
Contiene recursos estáticos:
- Imágenes
- Íconos
- Logos
- Fuentes


##  Cómo usar el proyecto

### Opción 1 — Uso simple

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
3. Navegar usando el menú.

---

### Opción 2 — Ejecutar directamente desde GitHub

1. Accedemos al repositorio creado en GitHub (Tienda-de-Tecnologia)
2. Accedemos al apartado de configuraciones del repositorio.
3. Visualizamos el apartado de pages en configuraciones.
4. Dar clic en el enlace disponible y accederemos al proyecto de forma automatica.
(https://elone1415.github.io/Tienda-de-Tecnologia/)
